# ScreenshotVB — Feature Roadmap

## V2 — Priority Features

- [ ] **Pin to screen** — keep screenshot floating on top of all windows for reference
- [ ] **Basic annotations** — draw arrows, boxes, and add text before copying/saving
- [ ] **Blur / redact tool** — blur sensitive info (passwords, emails) before sharing

## V2 — Medium Priority

- [ ] **Delay capture** — 3/5 second countdown before capturing (for menus, tooltips)
- [ ] **Window snap** — click on a window to capture just that window, no drag needed
- [ ] **Recent screenshots tray menu** — right-click tray icon to see last 5 screenshots
- [ ] **Scrolling capture** — capture full webpage/window by auto-scrolling

## V3 — Nice to Have

- [ ] **Custom hotkey** — let user change Ctrl+E to any key combo in settings
- [ ] **Auto-copy on capture** — skip preview window, copy straight to clipboard
- [ ] **Multi-monitor support** — pick which monitor to capture on

---

## Completed (V1)

- [x] Global hotkey Ctrl+E
- [x] Region select overlay (click & drag)
- [x] Auto-save to temp folder
- [x] Copy to clipboard (works with Paint, Word, Discord)
- [x] Save to custom folder (PNG/BMP/JPEG)
- [x] Drag & drop into any app (WhatsApp, Discord, Slack, folders)
- [x] Open temp folder in Explorer
